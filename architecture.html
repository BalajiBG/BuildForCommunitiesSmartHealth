✨ Platform Features

💊

Medicine Stock Monitoring

Real-time tracking with AI stock-out predictions, actionable indent system, expiry alerts

📊

Patient Insights (Power BI-style)

Interactive cross-filtering by department, age, gender, visit type — all-time analytics

🛏️

Bed Availability

Real-time tracking with timestamps, full-capacity alerts, patient diversion

👨‍⚕️

Doctor Attendance

Daily tracking, understaffed flagging (<50%), staff rotation directives

🤖

Agentic AI Assistant

Proactive recommendations, anticipates problems, drafts directives, navigates users to correct page/tab

🎤

Voice Input (STT)

Web Speech API for Hindi & English voice input. Users speak naturally, AI understands both languages

🔊

Text-to-Speech (TTS)

Read aloud AI responses in Hindi/English. Auto-detects language. Stop/play controls

🌐

Full Hindi Translation

100% UI in Hindi when selected. Static dictionary + Gemini live translation for dynamic content. Zero English visible

🔄

Smart Redistribution

AI recommends medicine/staff transfers: source → destination, quantity, explanation

📋

Admin Directives

Issue → Acknowledge → In Progress → Complete workflow. Auto-translated for Hindi staff

🏕️

Health Camps

Schedule, track, measure community outreach — target vs actual beneficiaries

🏥

Infrastructure Registry

Staff, lab equipment, facilities — full digital asset register per centre

📱

Contact Directory

One-tap call/email for referrals, emergency numbers, inter-centre coordination

📝

Audit Trail

Every action logged — user, time, category. Tamper-evident accountability

→

→
3 Gemini AI responds in user's language

📖

Static Dictionary

150+ keys translated to Hindi. Zero latency, offline-capable, instant UI switching

🔄

Gemini Live Translation

Dynamic content (directives, user-entered text) translated in real-time via Gemini 2.5 API

🎯

Context-Aware Language

Language preference persists per user. AI chat, voice, and all UI respect selected language
